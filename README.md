# Exploratory_Data_Analysis_E-commerce_Projects
Python-based Exploratory Data Analysis (EDA) projects analyzing e-commerce data. Covers data cleaning, customer segmentation (RFM), sales forecasting trends, and data visualization using Pandas, Seaborn, and Matplotlib.


## 📁 Projects Included
* **E-Commerce Summer Products Analysis:** Analyzing performance, ratings, and sales trends of e-commerce summer products.
* **Gold Price Analysis (1978-2023):** Visualizing and tracking historical trends in gold data.

## 🛠️ Tools & Libraries Used
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Automated EDA:** Sweetviz (Check out the generated `SWEETVIZ_REPORT.html`)
